Python

Python adalah bahasa pemrograman tingkat tinggi yang populer, mudah dipelajari, dan serbaguna (general-purpose), sering digunakan untuk data science, AI, web development, dan otomatisasi. Dibuat oleh Guido van Rossum pada 1991, Python dikenal dengan sintaks sederhana yang mirip bahasa Inggris, menjadikannya pilihan utama pemula. JavaScript

JavaScript adalah bahasa pemrograman tingkat tinggi, dinamis, dan berbasis objek yang digunakan untuk membuat halaman web menjadi interaktif dan dinamis. Sebagai salah satu teknologi inti web bersama HTML dan CSS, JS berjalan di sisi klien (browser) untuk menangani animasi, konten dinamis, dan respons pengguna tanpa perlu memuat ulang halaman.

Java

Java adalah bahasa pemrograman tingkat tinggi, berorientasi objek, dan berbasis kelas yang dirancang untuk memiliki ketergantungan implementasi seminimal mungkin. Dengan slogan "Write Once, Run Anywhere" (WORA), kode Java dapat berjalan di berbagai platform tanpa perlu dikompilasi ulang. Java sangat populer digunakan untuk pengembangan aplikasi backend web, Android, big data, dan enterprise software.

C#

C# (dibaca "C Sharp") adalah bahasa pemrograman modern, berorientasi objek (object-oriented), dan type-safe yang dikembangkan oleh Microsoft. Bahasa ini berjalan di atas platform .NET, bersifat open-source, serta mendukung pengembangan berbagai aplikasi lintas platform (cross-platform) seperti web, desktop, mobile, hingga game.
